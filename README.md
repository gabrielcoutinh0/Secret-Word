## ⚙️ Tecnologias

Este projeto foi desenvolvido utilizando:

## 📌 Começando

Clone o projeto:

```bash
git clone https://github.com/gabrielcoutinh0/Secret-Word
cd "Secret Word"
```

Instale as dependências :

```bash
npm i
npm run dev
```
